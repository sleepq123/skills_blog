### 了解概念
#### 什么是物理像素、逻辑像素、设备像素比？
##### 物理像素： 设备屏幕实际拥有的像素点。
> 如： ipone6的屏幕在宽度有750个像素点，高度有1334个像素点

##### 逻辑像素： 设备独立像素，也就是我们经常使用的px（全称：Device Independent Pixel）
> 如：
```css
.box{
    width: 300px;
    height: 200px;
}
```

 ##### 设备像素比（Device Pixel Ratio 简称：DPR）： 物理像素 / 逻辑像素 = 设备像素比
